The 34th Character

The 34th character (typed as X) is the vowel extender. When placed after a vowel, it extends that vowel into its long form:

  • a + X = aa
  • e + X = ey
  • i + X = ee
  • o + X = oa
  • u + X = oo

Sukun

In Vinara, sukun is written using the 35th character, typed as Shift + q. When placed after certain consonants at the end of a word, it indicates no vowel follows (implied sukun).

  • W + Q = h (glottal stop, އް)
  • s + Q = s (ސް)
  • n + Q = n (ން)
  • T + Q = th (ތް)
  • S + Q = sh (glottal stop, ށް)

In transliteration, you can type hh to specify ށް instead of އް.
